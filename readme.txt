Astronomy tools.

Simple utility (stars.py) to:

* compute local sidereal time
* draw a star map

Contains also an utility to process a star file in binary form for astro iq program

stars500 - 500 brightest stars (alpha, delta, mag)
stars.txt - 100k brightest
(HIP catalog number, alpha, delta, distance (AL), magnitude, type)

alpha & delta are in degrees
